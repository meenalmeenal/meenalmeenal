## 🚀 Featured Projects

### 🎓 GradeSmart — AI Answer Sheet Evaluator
*LLM-powered automated grading pipeline*
- **AI Core:** Llama 3.3 70B via Groq API with prompt-engineered semantic evaluation against model answer keys.
- **Tech:** FastAPI backend + React frontend with dynamic paper schema parsing (flat questions, subparts, sections).
- **Features:** Per-question confidence scoring, keyword extraction, structured JSON output with automated review flagging for teachers.

### 🧴 Skin Concern Detection Model using LLM & CV
*Deep learning system for facial skin analysis*
- **Features:** Detects acne, PIH, and other skin conditions from facial images using OpenCV preprocessing & augmentation.
- **AI Layer:** LLM + RAG-based skincare recommendation engine suggesting ingredients and products.
- **Interface:** Minimal UI for image upload and automated analysis (in development).

### 📖 Children's Story Text-to-Image Generator
*Full GenAI pipeline for visual storytelling*
- **Pipeline:** Text extraction from PDFs → scene-wise summaries via Ollama → structured JSON prompts → SDXL image generation.
- **Tech:** PyPDF, ChromaDB + SentenceTransformers for contextual scene retrieval via RAG workflow.

### 🧠 Personality Prediction — Big Five OCEAN Model
*Efficient ML model for personality trait prediction*
- **Innovation:** Reduced traditional 50-question test to 25 key questions via feature selection — same accuracy, half the effort.
- **Tech:** FastAI, LazyRegressor, LazyClassifier for automated model selection across multiple ML algorithms.
- **Use Cases:** HR screening, psychology applications, personalized recommendations.
